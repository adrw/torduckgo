TorDuckGo

Search the Onion version of DuckDuckGo by default in Tor Browser!

  1. Open this page in Tor Browser
  2. Click the green + in the search box
  3. Add TorDuckGo to your search options
  4. Set TorDuckGo as your default browser in Preferences -> Search
  5. Enjoy default searching DuckDuckGo with 3g2upl4pq6kufc4m.onion
